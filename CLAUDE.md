# Beer Tour

Next.js app. Uses pnpm and mise (Node 22).

## Setup

In worktrees, run `pnpm install` from both the repo root (`/Users/henric/beer-tour`) and the worktree directory before starting the dev server.

## Commands

- `pnpm dev` — start dev server on port 3000
- `pnpm build` — production build
- `pnpm lint` — run ESLint
